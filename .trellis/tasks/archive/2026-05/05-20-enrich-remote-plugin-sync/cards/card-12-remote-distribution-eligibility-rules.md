# Card 12: Remote Distribution Eligibility Rules

## Goal

明确哪些条目可对某台机器分发，以及为什么不可分发。

## Scope

* 远程同步声明
* 平台匹配
* 标签和策略匹配
* 不可分发原因枚举

## Acceptance

* 平台可稳定判断条目对目标机器是否可分发
* 仓页可显示不可分发原因
* 默认视图只展示可分发条目

## Suggested Implementation Steps

1. 明确可分发判定所需输入
2. 枚举不可分发原因
3. 建立统一判定函数
4. 建立“默认只看可分发条目”的过滤逻辑
5. 建立“查看不可分发条目”的原因展示结构
6. 增加判定规则回归测试

## Done Means

* 平台和页面共享同一套判定逻辑
* 可分发与不可分发原因都可解释
* 不再依赖页面层拼判断
