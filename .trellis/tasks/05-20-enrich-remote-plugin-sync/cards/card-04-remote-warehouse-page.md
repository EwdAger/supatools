# Card 04: Remote Warehouse Page

## Goal

建立远程插件仓主页面，并与单服务器批量同步合并。

## Scope

* 未选服务器时的仓总览
* 服务器选择器
* 仓条目排序
* 选定服务器后的默认可分发视图
* “查看不可分发条目”切换

## Acceptance

* 未选服务器时能查看仓总览和仓条目更新状态
* 选定服务器后只默认展示可分发条目
* 可切换查看不可分发条目及原因

## Suggested Implementation Steps

1. 确定远程插件仓页的路由与菜单入口
2. 实现未选服务器状态
3. 实现仓总览默认排序
4. 实现“选定服务器后”的页面切换
5. 实现“查看不可分发条目”切换与原因展示
6. 实现仓条目级动作位
7. 明确仓页与远程 Agent 页之间的跳转关系
8. 增加页面状态切换与条目过滤测试

## Done Means

* 远程插件仓页可在“仓总览”和“服务器上下文”之间稳定切换
* 默认视图聚焦正确
* 不可分发原因可被查看
