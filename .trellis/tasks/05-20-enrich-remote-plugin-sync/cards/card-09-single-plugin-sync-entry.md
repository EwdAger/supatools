# Card 09: Single Plugin Sync Entry

## Goal

在插件详情里提供单插件“同步到远程”能力。

## Scope

* 插件详情顶部动作入口
* 插件详情二级页选择服务器
* 只展示已配置服务器
* 同步摘要展示

## Acceptance

* 插件详情可进入“同步到远程”二级页
* 二级页可对指定服务器发起同步
* 插件详情主面板不承担配置编辑职责

## Suggested Implementation Steps

1. 在插件详情工具栏增加“同步到远程”入口
2. 增加插件详情二级页
3. 明确服务器列表筛选规则
4. 展示最小必要的同步前状态
5. 对不可同步服务器给出轻量处理

## Done Means

* 插件详情具备单插件同步远程能力
* 二级页职责清楚，不侵占机器配置和完整状态职责
