# Card 10: Manual Sync Engine Upgrade

## Goal

把同步语义收敛成显式的安装、升级、配置下发。

## Scope

* 单服务器批量同步
* 单插件同步
* 不自动卸载
* 版本差异与配置状态处理

## Acceptance

* 可对单台机器执行批量同步
* 同步只覆盖安装、升级、配置下发
* 不会自动卸载远端多余插件

## Suggested Implementation Steps

1. 统一单插件同步与批量同步的动作语义
2. 明确同步计划的输入
3. 明确安装 / 升级 / 配置下发的执行顺序
4. 明确单服务器批量同步的选中模型
5. 明确同步结果回写和状态刷新策略
6. 确保不会自动卸载远端多余插件
7. 增加单台机器同步流程测试

## Done Means

* 同步引擎语义固定
* 单插件和批量同步走同一底层规则
* 不会隐式做超出范围的动作
