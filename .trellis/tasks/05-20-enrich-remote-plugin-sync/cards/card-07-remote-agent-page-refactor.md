# Card 07: Remote Agent Page Refactor

## Goal

把远程 Agent 页改造成小窗友好的列表 -> 详情结构。

## Scope

* 默认机器列表页
* 新增机器二级页
* 机器详情二级页
* 机器详情中的安装命令、插件状态、配置、日志

## Acceptance

* 默认主视图不同时展开创建、列表、详情三层信息
* 点击机器进入详情页
* 机器详情页可编辑机器-插件配置并查看安装状态

## Suggested Implementation Steps

1. 把当前远程 Agent 页拆成三个视图状态
2. 调整默认入口
3. 把“新增机器”迁移成二级页
4. 把“机器详情”迁移成二级页
5. 在详情页中重新组织内容顺序
6. 确保小窗模式下内容密度合理

## Done Means

* 远程 Agent 页符合小窗使用心智
* 列表/新增/详情职责分离
* 不再默认挤压三层信息
