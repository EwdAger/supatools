# 2.5.0

## 新功能 (Feat)

- 新增远程插件同步基础能力，补齐插件远程分发元数据、可分发规则和远程仓快照链路
- 新增远程 Agent 运行时基础，支持远端机器接入、安装状态采集、机器级插件配置与同步日志
- 新增远程插件仓页面，支持仓总览、单机视角筛选及远程插件批量同步
- 远程 Agent 安装流程支持 `systemctl` 优先、`nohup` 回退，并支持按平台标签选择安装后置脚本模板

## 修复 (Fix)

- 修复远程插件同步只依赖本地安装插件的问题，批量同步改为以远程插件仓快照为唯一分发来源
- 修复远程批量同步会误触发远端卸载的问题，同步语义收敛为安装、升级和配置下发
- 修复 pre-commit 在未暂存文件上执行格式化的问题，限制为只处理 staged 文件

## 优化 (Optimize)

- 重构远程 Agent 设置页为更适合小窗口的列表、创建页、详情页切换流

## 重构 (Refactor)

无

## 其他 (Chore)

- 引入项目级 Trellis 工作流基线，补齐 Codex、Claude、OpenCode 平台集成与任务记录
